GarminBot

Purpose: Telegram bot that syncs Garmin Connect health/activity data daily, sends formatted reports with weekly summaries, exercise charts, and integrates nutrition tracking with Groq LLM-powered recommendations.
Pythonpython-telegram-botGarmin Connect APIGroq LLMSQLiteDockerMatplotlib

Architecture Overview

Key Workflows

Daily Sync (08:00 UTC)
Scheduler triggers → garmin_sync.py authenticates with Garmin Connect → fetches steps, HR, sleep, calories, workouts → stores in SQLite → generates Matplotlib charts → Telegram bot sends formatted daily report to subscriber.
Weekly Summary (Sundays)
Aggregates 7-day data from SQLite → calculates trends → Groq LLM generates personalized recommendations → sends rich report with charts to Telegram.
Nutrition Tracking
User sends meal via Telegram → bot processes natural language → logs macros in SQLite → tracks against daily goals → reports caloric deficit/surplus.

Components

  • garmin_sync.py — Authenticates and fetches data from Garmin Connect
  • bot.py — Telegram bot with commands (/today, /week, /nutrition)
  • chart_gen.py — Matplotlib chart generation for activity trends
  • scheduler — APScheduler for daily/weekly cron tasks
  • SQLite DB — Persists activity, nutrition, and subscriber data
  • Groq API — LLM-powered workout and nutrition recommendations
