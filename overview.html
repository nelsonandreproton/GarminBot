Daily fitness sync, nutrition tracking, and workout recommendations via Telegram

1. What this project does

Purpose

Telegram bot that authenticates with Garmin Connect daily, fetches health metrics (sleep, steps, calories, HRV), stores them in SQLite, and sends formatted reports in Portuguese. Also handles nutrition logging via barcode/text and generates AI-powered workout plans using Groq.

3. Scheduled jobs

07:00 daily

Garmin Sync

Authenticate → fetch sleep quality, steps, calorie burn, HRV, training readiness → upsert to SQLite with exponential backoff on 429.

08:00 daily

Daily Report

Query yesterday from DB → format in Portuguese → send via Telegram with emoji indicators for sleep, steps, and caloric balance.

Sunday 20:00

Weekly Summary

Aggregate 7 days → compute trends (streak, weekday vs weekend) → generate Matplotlib bar chart → send with Groq-written insights.
